Detailed Description

Solve quadratic programming problems using an active set approach.

QP formulation: min 0.5 z^T G z + g^T z s.t lbineq <= AiZ z <= ubineq

The matrix G is specified in terms of the inverse of its Cholesky decomposition (Li): G = LL^T Li = inv(L)
